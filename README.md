# Developer-Focused Projects

Three full-stack web development projects — built with React, Node.js, and TailwindCSS.

---

## Project 1 — GitHub Activity Tracker

🔗 **Branch:** https://github.com/malikibraheem2023-spec/Developer-focused/tree/github-tracker
🌐 **Live Demo:** https://malikibraheem2023-spec.github.io/Developer-focused/

Search any GitHub username and instantly see their full developer profile — language breakdown charts, commit activity calendar, top repositories, and a shareable portfolio card. Includes Google-style search suggestions with popular developer names.

---

## Project 2 — Mini Deploy Dashboard

🔗 **Branch:** https://github.com/malikibraheem2023-spec/Developer-focused/tree/deploy-dashboard

Add your servers through a UI form, check live SSH status (Online/Offline), trigger deployments with a single button, and watch real-time logs stream directly in the browser terminal.

---

## Project 3 — URL Shortener + Analytics

🔗 **Branch:** https://github.com/malikibraheem2023-spec/Developer-focused/tree/url-shortener

Paste any long URL and get a short link instantly. Every click is tracked — country, traffic source (WhatsApp / Google / Direct), and time — with analytics charts per link.

---

**Stack:** React · Vite · TailwindCSS · Node.js · Express · Socket.io · SQLite · Chart.js
