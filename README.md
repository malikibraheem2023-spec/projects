# Developer-Focused Projects

Three mid-level web development projects built with React + Vite (frontend) and Node.js + Express (backend).

## Projects

| Project | Branch | Description |
|---|---|---|
| GitHub Activity Tracker | `github-tracker` | Search any GitHub user — visualize repos, languages, commits, and generate a shareable portfolio |
| Mini Deploy Dashboard | `deploy-dashboard` | Add servers via UI, stream live logs, trigger deploys over SSH |
| URL Shortener + Analytics | `url-shortener` | Shorten URLs and track click count, country, and referrer per link |

## Stack

- **Frontend:** React + Vite + TailwindCSS
- **Backend:** Node.js + Express
- **Database:** PostgreSQL (URL Shortener), JSON store (Deploy Dashboard)
- **Real-time:** Socket.io (Deploy Dashboard logs)

## Getting Started

Each project lives on its own branch. Switch to the branch and follow the README inside that folder.

```bash
git checkout github-tracker
git checkout deploy-dashboard
git checkout url-shortener
```
