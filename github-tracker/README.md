# GitHub Activity Tracker

Search any public GitHub user — view their repos, languages, recent activity, and export a standalone portfolio HTML file.

## Features

- Search by GitHub username (unauthenticated, no token required)
- User profile card (avatar, bio, stats)
- Language breakdown doughnut chart
- Top repositories by star count
- Recent public activity feed
- One-click portfolio HTML export

## Stack

- React + Vite
- TailwindCSS
- Chart.js / react-chartjs-2
- GitHub REST API (public, unauthenticated)

## Run

```bash
npm install
npm run dev
```

App runs at http://localhost:5173
